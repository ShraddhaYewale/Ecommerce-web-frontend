# ShopNow Frontend - Modern E-Commerce Interface

A beautiful, responsive, and user-friendly **frontend** for an e-commerce web application built with **HTML, CSS, Bootstrap 5, and Vanilla JavaScript**.

## ✨ Key Features

- Modern & responsive design (mobile-first)
- Home, Categories, Shop, Product Detail, Cart, Checkout flows
- User Account section: Profile, Orders, Wishlist, Settings (with form validation)
- Persistent shopping cart using localStorage
- Category cards loaded dynamically from JavaScript array
- Hover effects, smooth transitions, clean typography
- Ready for future backend integration (REST API calls)

## 🛠️ Tech Stack (Frontend only)

- **HTML5** + **CSS3**
- **Bootstrap 5.3** (responsive grid, components, utilities)
- **Vanilla JavaScript** (cart logic, form validation, dynamic rendering)
- **Font Awesome 6** (icons)
- **LocalStorage** (temporary cart & wishlist persistence)
